O que é elemento de formulario?

O elemento form é a tag form que tem a tag de abertura e fechamento chamada 'form'

Entre o inicio e o final da tag podemos incluir outros elementos que servem para os usuarios entrarem com informações

Onde os dados são processados?

Quando um dado é enviado(quando o usuario clica no botão de enviar) por exemplo, o navegador faz uma requisição e um script pode responder a essa requisição e processar os dados

Como os dados são transferidos?

Por padrão, os dados do formulario são enviados com uma requisição get, com os dados enviados anexados na URL

se enviarmos 'frog' no exemplo acima, o navegador fará uma requisição com a seguinte URL

//example.com/animals?animal=frog

neste caso, podemos acessar os dados com o backend ou com o frontend

se quisermos, podemos instruir com que o formulario use o methodo post

<form method="post">
...
</form>

desta forma, os dados irão no corpo da requisição

Qual método usar?

Para dados sensiveis usa-se o post, pois os dados serão criptografados(se estivermos usando o método HTTPS) e só será acessivel pelo script backend que processara a requisição
